iRes Embed Demo

Widget + Buttons op 1 pagina

Deze demo laat direct zien hoe je iRes correct embedt: eerst de controller bootstrap, daarna je <ires-button> en optioneel <ires-widget>.

Hoe inbouwen

  1. Plaats 1x bootstrap in je pagina: ires-config + script.
  2. Plaats daarna je <ires-button src="..."> tags.
  3. Gebruik alleen <ires-widget> als je widget-mode wil gebruiken.

Bootstrap (eenmalig op pagina)

<div id="ires-config"
     ires-config="https://default.i-reserve.net/api/rest/embed/30120"
     hidden></div>
<script type="module"
     src="https://default.i-reserve.net/javascript/embed/iResController.js"></script>

Live demo

Knoppen hieronder zijn live. `30128`/`30132` openen modal of widget afhankelijk van hun JSON type.

Deze widget is optioneel. Laat hem staan als je ook type: "widget" knoppen gebruikt.

Snippet: Alleen modal buttons

<div id="ires-config"
     ires-config="https://default.i-reserve.net/api/rest/embed/30120"
     hidden></div>
<script type="module"
     src="https://default.i-reserve.net/javascript/embed/iResController.js"></script>

<ires-button src="https://default.i-reserve.net/api/rest/embed/30128"></ires-button>
<ires-button src="https://default.i-reserve.net/api/rest/embed/30132"></ires-button>

Snippet: Modal + Widget mode

<div id="ires-config"
     ires-config="https://default.i-reserve.net/api/rest/embed/30120"
     hidden></div>
<script type="module"
     src="https://default.i-reserve.net/javascript/embed/iResController.js"></script>

<ires-widget id="main-widget"
     src="https://default.i-reserve.net/api/rest/embed/30120"></ires-widget>
<ires-button src="https://default.i-reserve.net/api/rest/embed/30128"></ires-button>
